# Review Report 内部 Finding 契约

每个 `REV-*` 包含 severity、responsible_document、responsible_item、owner_skill、impact_set、review_scope、close_conditions、close_evidence 和 status。Finding 不创建独立 Markdown。BLOCKER/HIGH 未关闭时报告 blocked。

关闭必须满足：责任文档已修复；影响集重新计算；close conditions 非空；每个 close evidence 是真实可读产物；受影响范围已局部复核。仅写“已修复”不能关闭。
