# Novel World Skill

用于在不打断主流程的前提下，逐步补充世界规则与背景信息。

## 定位

1. 可选扩展阶段，按需补充。  
2. 用户未主动要求时，不强制展开世界观工程。  
3. 若当前正在推进章节创作，世界观补充应“最小可用”。  

## 输出文件

`01-世界观/world-overview.md`（可逐步拆分子文档）

```markdown
# 世界观总览（可选扩展）

## 当前创作必需规则
- 规则1：
- 规则2：

## 已确定背景
- 时间背景：
- 地域背景：
- 势力背景：

## 暂缓细化
- [后续再补，不阻塞当前章节]
```

## 规则

1. 只补当前剧情需要的设定，避免一次性过度设计。  
2. 所有新增设定不得与已定稿章节冲突。  
3. 表达直接、可执行，不使用文学修辞包装规则。
