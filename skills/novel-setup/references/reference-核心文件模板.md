# 核心文件模板

## 1. 故事大纲.md

```markdown
# 故事大纲

## 一句话梗概
[30字内，故事的灵魂]

## 核心冲突
- 主要冲突：[主角 vs X，具体什么问题]
- 深层冲突：[价值观/身份/认知层面的对抗]

## 主题
[这个故事想表达什么，1-2句]

## 整体结构（按卷/幕划分）

### 第一卷：[卷名]（预计第1-X章）
- 核心事件：[卷级事件]
- 主角变化：[起点→终点]
- 关键章节：
  - 第1章：[一句话]
  - 第2章：[一句话]
  - ...
- 本卷末钩子：[引向下一卷的悬念]

### 第二卷：[卷名]
（同上）

## 主线与支线
- 主线A：[名称] — 状态：[推进中/待启动/已完结]
- 支线1：[名称] — 状态：...

## 结局方向
[大致走向，可多版本并存]
```

## 2. 主角档案.md

```markdown
# 主角档案：[姓名]

## 基本画像
- 年龄/外貌/身份：
- 性格底色：[3-5个关键词]
- 口头禅/动作习惯：

## 成长轨迹（弧光）
- 起点状态：[故事开始时的内在状态]
- 转折节点：
  - 节点1：[什么事件引发什么变化]
  - 节点2：...
- 终点状态：[故事结束时期望达到的状态]

## 当前状态（随剧情更新）
- 当前章节：第X章
- 等级/力量：
- 心境：
- 主要目标：[短期]
- 深层渴望：[长期]
- 最大恐惧/软肋：

## 重要人际关系
- [角色A]：关系类型 — 当前状态
- ...

## 主角已知/未知清单（核心真相）
- ✅ 已知：[关键认知点1] — 何时获知
- ❌ 未知：[关键真相1] — 预计何章揭开
```

## 主角当前状态

## 4. 伏笔追踪.md

```markdown
# 伏笔追踪

## 图例
- 🟢 已回收　🟡 进行中　🔴 待回收　⚫ 已废弃

## 伏笔列表

### F001 [伏笔简称]
- 状态：🔴 待回收
- 埋设位置：第X章（具体段落/事件）
- 埋设方式：[直述/暗示/道具/对白]
- 计划回收：第Y章左右
- 回收方式：[预想的揭晓方式]
- 关联设定：[相关实体/规则]
- 备注：

### F002 ...
```

## 5. 当前冲突.md（短期核心，每章定稿后审视，每5章必更新）

```markdown
# 当前冲突

> 最后更新：第X章定稿后
> 本冲突起始于：第Y章
> 预计延续至：第Z章左右

## 一句话冲突
[主角 vs 什么 / 必须解决什么 / 否则会怎样，30字内]

## 冲突结构

### 表层冲突（眼下要解决的事）
- 对手/障碍：[具体的人/物/局面]
- 主角的目标：[要达成什么]
- 时限/压力：[多久内必须出结果，不解决会怎样]

### 深层冲突（驱动表层的内核）
- 价值观/身份层面：[主角内在的撕扯]
- 与主角弧光的关系：[本冲突如何推动主角的成长节点]

## 双方筹码与风险
- 主角持有：[资源、信息、关系、能力]
- 对手持有：[威胁、信息差、资源优势]
- 主角软肋（对手可能利用）：
- 对手软肋（主角可能利用）：

## 推进刻度（关键节拍）
> 用于判断每章是否在"啃这块骨头"

- [ ] 节拍 1：[事件/认知/对抗] — 预计第 Y+? 章
- [ ] 节拍 2：...
- [ ] 节拍 3：...
- [x] 已完成节拍：第 ? 章 [简述]

## 与其他文件的接口
- 关联伏笔（来自 `伏笔追踪.md`）：F00X、F00Y
- 涉及主角弧光节点（来自 `主角档案.md`）：[节点名]
- 涉及活跃支线（来自 `故事状态.md`）：[支线名]

## 演化预案
- 若主角达成 → 下一阶段冲突可能是：[简述]
- 若主角失败 → 下一阶段冲突可能是：[简述]
- 若中途转化 → 走向：[简述]

## 并行冲突（如存在）
- 冲突B（支线/并发）：[简述] — 状态：[活跃/待触发/已归档]
  - 起始于：第Y章
  - 当前节拍：[简述]
  - 与主线冲突的交互：[促进/牵制/转化]
- ...

## 历史冲突链（按时间倒序）
- 第A章 — 第B章：[上一轮冲突简述] → 结局：[达成/失败/转化]
- ...
```

## 6. 写作注意事项.md

```markdown
# 写作硬性约束

## 视角规则
- 主视角：[第一/第三限制/全知]
- 视角切换规则：[允许/禁止/限定条件]

## 信息对称规则
- 读者认知 ≤ 主角认知（或相反）
- 主角未知的核心真相禁止在正文直接展示

## 文风硬约束
- 禁止出现：[如现代梗、网络用语、过度bl描写等]
- 必须保持：[如古风白、冷峻调、幽默感等]

## 其他
- 章节长度目标：[如 2500-4000 字]
- 对白/描写比例偏好：
```
