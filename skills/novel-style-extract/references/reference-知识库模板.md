# 知识库文件模板

## 维度1：总纲提取（全量时必做，增量时可选）

```markdown
# 创作风格总纲

## 文风灵魂（一句话）
[从参考小说整体语感提炼]

## 核心铁律（3-5条）
1. [从参考小说中反复出现的约束提炼]
2. ...

## 参考坐标
- 形似但禁止：[不希望被误认为模仿的作品/作者]
- 神似目标：[追求其境界的作品/作者]

## 词汇基调
- 偏好词族：[从参考小说高频词和特色用词提炼]
- 慎用词族：[参考小说中极少出现或避免使用的词类]
- 禁用词清单：[参考小说中绝对未出现的词，或用户要求禁止的词]

## 句式偏好
- 长短句比例倾向：[统计参考小说典型段落]
- 标点偏好：[如参考小说偏好破折号/句号/省略号等]
```

## 维度2：场景分片提取（核心）

针对每种场景类型，从参考小说中找出**最典型的2-3段**，进行结构化分析：

```markdown
# 场景描写：[场景类型]

## 目标
[从参考小说该类场景的写作效果提炼]

## 必做（参考小说中反复出现的技法）
- [ ] [技法1：参考小说中具体出现的写法]
- [ ] [技法2：...]

## 必避（参考小说中明确不用的写法）
- [ ] [反技法1：与参考小说风格冲突的写法]
- [ ] [反技法2：...]

## 量化指标（从参考小说统计得出）
| 项目 | 不合格 | 合格 | 优秀 |
|------|--------|------|------|
| [指标1] | <X | X-Y | ≥Z |
| ... | | | |

## 正面示例（来自参考小说）
> [引用参考小说中最典型的1-2段]

**为什么好**：[结合"必做"和"量化指标"逐项点评]

## 反面示例（与参考小说风格冲突的虚构例子）
> [构造一段明显违背参考小说风格的文字]

**问题**：
- 违反必做第X条
- 量化指标项Y不达标
```

## 维度3：自检清单提取（全量时必做，增量时同步更新）

```markdown
# 写作风格自检清单

> 用法：每写完一章正文（或一次重大修改后），逐项检查并打分。
> 总分 ≥85 / 100 视为风格合格；<70 必须返工；70~85 视为可改进。

## 一、铁律检查（每条违反 -10 分，0 容忍）

- [ ] 总纲铁律1：[从总纲提取]
- [ ] 总纲铁律2：...
- [ ] 视角铁律：...
- [ ] 信息对称铁律：...
- [ ] 禁用词清单：0 命中

## 二、场景执行检查（按本章涉及的场景类型，每类满分 10）

### [场景类型]（从对应分片提取必做项和量化指标）
- [ ] 必做项 X 条全部命中（每漏 1 项扣 2 分）
- [ ] 量化指标：[具体指标项]
- 实际得分：___ / 10

## 三、文风指纹比对（10 分）
...（标准模板，见 novel-setup）

## 四、词汇与句式（10 分）
...（标准模板，见 novel-setup）

## 五、信息密度与节奏（10 分）
...（标准模板，见 novel-setup）

## 六、量化总评
...（标准模板，见 novel-setup）
```
