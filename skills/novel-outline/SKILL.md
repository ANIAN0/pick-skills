# Novel Outline Skill

用于补充全书宏观结构，帮助长期连载时做风险控制和伏笔规划。

## 定位

1. 可选扩展，不是刚需前置。  
2. 只在用户主动要求“写整体大纲/重构全局结构”时调用。  
3. 不得反向要求用户先补全世界观/角色设定才能继续主流程。  

## 前置条件

建议满足（非强制）：
- `topic_confirmed = true`
- `chapter1_final_confirmed = true`
- `volumes_confirmed = true`

## 输出文件

`03-大纲/outline.md`

```markdown
# 全书大纲（可选扩展）

## 核心主线
- 起点：
- 中段关键升级：
- 终局方向：

## 与已确认内容对齐
- 已确认选题：
- 已确认三句话故事：
- 第1章正文锚点：
- 分卷结构锚点：

## 风险与回收
- 需要回收的伏笔：
- 可能失控的支线：
- 节奏风险点：
```

## 规则

1. 大纲必须服从已确认内容，不得推翻已定稿结论。  
2. 表达保持直接，避免空泛术语堆砌。  
3. 大纲只能“辅助推进”，不能替代故事单元与逐章机制。
