# 必看文档模板（阶段2）

基于选定创作方向，收集本章创作所需全部资料，浓缩为一份文档。

**这是整个4阶段流程的关键环节**：必看文档是阶段3创作的**唯一依据**。创作正文时不加载分散的设定文件，只参照这份文档。

---

## 处理步骤

### 步骤1：加载输入文件

| 类型 | 文件 | 加载条件 |
|------|------|---------|
| 简报与方向 | `00_开写简报.md`、`01_创作方向.md` | 必须 |
| 实体设定 | `相关设定/角色/*.md` | 本章出场角色 |
| 实体设定 | `相关设定/物品/*.md` | 本章涉及物品 |
| 实体设定 | `相关设定/技能/*.md` | 本章可能使用的技能 |
| 待定设定 | `工作区/待定设定/_索引.md` | 检查是否有本章相关 |
| 待定设定 | `工作区/待定设定/<类别>/<名称>.md` | 有相关时加载 |
| 上章衔接 | `正文/第X-1章/正文.md` | N>1时，加载结尾部分 |
| 上章衔接 | `正文/第X-1章/事件摘要.md` | N>1时 |
| 上章必看 | `正文/第X-1章/_预创作/02_必看文档.md` | N>1时，可选参考 |
| 风格总纲 | `相关设定/创作风格/_总纲.md` | 必须 |
| 风格分片 | `相关设定/创作风格/场景描写/<场景>.md` | 本章预计场景类型 |
| 自检清单 | `相关设定/创作风格/自检清单.md` | 提取自测项 |
| 上章指纹 | `相关设定/创作风格/风格指纹/第X-1章.md` | N>1时 |

**第1章特殊处理**：无上章衔接，此项替换为`主角档案.md`起点状态 + `故事状态.md`初始快照。

### 步骤2：提取设定约束

从实体设定中提取本章必须遵循的约束：

| 约束类型 | 来源 | 提取内容 |
|---------|------|---------|
| 主角知晓边界 | `主角档案.md`的已知/未知清单 | 列出主角"未知"的真相，严禁在正文展示 |
| 角色行为约束 | `相关设定/角色/*.md` | 性格底色、行为倾向、关键限制 |
| 能力边界 | `相关设定/技能/*.md` | 代价、冷却、副作用、触发条件 |
| 规则边界 | `相关设定/世界设定.md` | 涉及规则的限制、触发条件 |

### 步骤3：提取风格要求

从创作风格知识库中提取：

| 来源 | 提取内容 |
|------|---------|
| `_总纲.md` | 核心铁律、词汇基调、句式偏好、禁用词 |
| `场景描写/*.md` | 对应场景的必做、必避、量化指标 |
| `自检清单.md` | 自测项模板 |

**⚠️ 风格分片缺失处理**：
- 若发现某场景分片缺失 → 在必看文档标注 `[场景X分片待补充，按通用规范处理]`
- 建议用户调用novel-style-extract补充

### 步骤4：提取衔接要点

从上一章结尾提取（N>1时）：

| 要点 | 来源 |
|------|------|
| 主角状态 | 上章正文结尾：位置/情绪/伤情/携带物 |
| 悬念钩子 | 上章正文结尾留下的未决问题 |
| 在场人物 | 上章结尾时谁在主角身边 |
| 时间环境 | 时间流逝/地点/天气 |

从开写简报提取：

| 要点 | 来源 |
|------|------|
| 冲突节拍 | 简报中的"本章应推进的下一节拍" |
| 伏笔动作 | 简报中的"伏笔动作建议" |
| 弧光推进 | 简报中的"主角弧光节点" |

### 步骤5：构建自测逻辑

将风格要求和设定约束转化为可执行的自测项：

| 自测类型 | 来源转化 |
|---------|---------|
| 铁律自测 | 总纲铁律 → 逐条检查项 |
| 场景执行自测 | 场景必做 → 勾选项；量化指标 → 数值填空 |
| 设定遵循自测 | 知晓边界 → "是否越界"；能力边界 → "是否交代代价" |
