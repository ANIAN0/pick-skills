# 自检报告模板（阶段3）

写完正文后，**必须**逐项自测并写入 `工作区/第X章VX/自检报告.md`。

## 创作流程（阶段3）

**此阶段只加载两份文档**：`02_必看文档.md` 和 `01_创作方向.md`（含用户决策）。不再重新读取分散的设定文件。

1. 读取 `02_必看文档.md`（唯一创作依据）
2. 读取 `01_创作方向.md`（确认用户选择的方向）
3. 按必看文档中的"创作方向确认"和"衔接要点"开始写作
4. 写作过程中参照：设定约束、风格要求、结构约束
5. **安全阀**：写作中如需使用必看文档未覆盖的设定/规则/实体，**立即停止创作**，标注：
   > 「安全阀触发：需补充必看文档——涉及 [实体/规则名称]，当前必看文档未收录其 [约束/代价/边界]。请回退到阶段2补充后再继续。」
   禁止凭记忆"补充"设定，防止设定漂移。
6. 写完正文后，**逐项检查"创作自测项"**

## 自测后续处理

- **≥85分**：可进入定稿流程
- **70~85分**：提示用户是否需要改进，列出主要失分项
- **<70分**：必须返工，列出必须修改的项
