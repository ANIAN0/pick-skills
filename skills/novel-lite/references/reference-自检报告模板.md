# 自检报告模板（阶段3）

写完正文后，逐项自测并写入 `工作区/第X章VX/自检报告.md`。

---

## 创作流程（阶段3）

**此阶段只加载两份文档**：
- `02_必看文档.md`（创作唯一依据）
- `01_创作方向.md`（用户决策）

**不重新读取分散的设定文件**。

### 创作步骤

1. 读取必看文档和方向文档
2. 按必看文档的"创作方向确认"和"衔接要点"开始写作
3. 写作过程中参照：设定约束、风格要求、结构约束
4. **安全阀机制**：
   - 如需使用必看文档未覆盖的设定 → **立即停止创作**
   - 标注：`「安全阀触发：需补充必看文档——涉及[实体/规则名称]，当前未收录。请回退阶段2补充。」`
   - 禁止凭记忆"补充"设定，防止设定漂移
5. 写完正文后，逐项检查自测项

### 自测后续处理

| 分数 | 处理 |
|------|------|
| ≥85分 | 可进入定稿流程 |
| 70~85分 | 提示用户是否改进，列出主要失分项 |
| <70分 | 必须返工，列出必须修改的项 |

## 自测执行检查清单

自测完成后检查：

- [ ] 铁律每条都有判定结果？
- [ ] 主角"未知"真相逐段检查无展示？
- [ ] 每个出场角色的行为符合设定？
- [ ] 每个使用的技能/能力交代了代价？
- [ ] 上章结尾状态自然衔接？
- [ ] 冲突节拍确实推进了？
- [ ] 伏笔动作完成了（回收/强化/新埋）？
- [ ] 本章涉及场景的必做项全部检查？
- [ ] 禁用词全文搜索确认0命中？
- [ ] 风格指纹偏差说明原因？
- [ ] 章末有钩子或情绪落点？
- [ ] 无水段（连续超80字纯氛围）？
