# 创作方向模板（阶段1）

基于开写简报，提出3个不同创作方向供用户选择。

---

## 处理要点

### 分析简报内容

从开写简报提取：
- 本章在大纲中的定位与预期事件
- 当前冲突状态与本章应推进的节拍
- 上一章结尾（衔接点）
- 伏笔动作建议（待回收/待强化/新埋）
- 主角弧光节点
- 约束提醒（视角、信息对称）
- 风险提示

### 方向差异维度

每个方向在以下维度中的**至少2个**形成差异：

| 维度 | 差异选项 | 示例 |
|------|---------|------|
| 叙事节奏 | 快/慢/紧/弛 | 快节奏推进事件 vs 缓节奏铺垫氛围 |
| 冲突强度 | 高压/渐升/反复拉锯 | 高压对抗 vs 持续施压渐升 |
| 信息释放 | 逐步揭示/集中爆发/侧面暗示 | 分段释放真相 vs 一次性揭晓 |
| 视角选择 | 内聚焦/外聚焦/交替 | 深挖主角内心 vs 侧重外部观察 |
| 情绪基调 | 压抑/爆发/悬疑/温情 | 压抑隐忍 vs 爆发宣泄 |
| 伏笔侧重 | 回收优先/新埋优先/强化优先 | 优先回收F001 vs 新埋伏笔种子 |

### 第1章特殊处理

第1章无"上一章衔接"，替换为：
- 故事起点切入方式：直接进入冲突 / 先铺氛围再引入主角 / 悬念钩子开场
- 承接：`主角档案.md` 起点状态 + `故事状态.md` 初始快照 + `当前冲突.md` 首段冲突

## 方向设计示例

### 示例：第5章（接第4章主角被背叛）

**简报关键信息**：
- 大纲定位：主角发现背叛者的真实目的
- 当前冲突节拍：节拍2（主角发现真相）
- 上章结尾：主角被背叛者刺伤，倒在荒野
- 伏笔动作：建议回收F001（背叛者的真实身份）
- 弧光节点：从"信任他人"到"开始怀疑"

**方向设计**：

```markdown
## 方向A：爆发型真相揭示
- 叙事节奏：快，密集事件推进
- 冲突强度：高压，主角发现真相后立即对抗
- 情绪基调：爆发，愤怒+决心
- 信息释放：集中爆发，一次性揭示背叛者全部阴谋
- 伏笔侧重：回收F001（身份揭示）+ 强化F002（某配角的真实立场）
- 预计场景类型：信息释放+对话+心理
- 与上一章衔接：主角醒来，立即开始追查
- 风险：节奏过快可能削弱情感铺垫

## 方向B：渐进型真相拼凑
- 叙事节奏：缓→紧，先铺垫后爆发
- 冲突强度：渐升，从受伤恢复到逐步发现线索
- 情绪基调：压抑→爆发，从虚弱到愤怒积累
- 信息释放：逐步揭示，主角从多个线索拼凑真相
- 伏笔侧重：回收F001（分两次揭示）+ 新埋F003（背叛者的幕后之人）
- 预计场景类型：探索+信息释放+心理
- 与上一章衔接：主角艰难求生，过程中发现线索
- 风险：节奏偏慢，需要足够的悬念维持

## 方向C：侧面暗示型真相
- 叙事节奏：弛→紧，先氛围后转折
- 冲突强度：反复拉锯，主角受伤→遇到援助→发现援助者也有目的
- 情绪基调：悬疑→震惊，先困惑后真相
- 信息释放：侧面暗示，通过其他角色反应揭示真相
- 伏笔侧重：强化F002（某配角的态度变化）+ 回收F001（间接揭示）
- 预计场景类型：对话+心理+信息释放
- 与上一章衔接：主角被"救"，但救他的人也有秘密
- 风险：读者可能比主角先知道真相，需注意信息对称

## 推荐：方向B
- 原因：简报要求推进节拍2（发现真相），方向B的"逐步拼凑"最贴合节拍要求；同时允许情感铺垫，符合弧光节点"从信任到怀疑"的转变过程
```

## 方向差异检查清单

自检每个方向是否形成足够差异：

- [ ] 至少2个维度形成明显差异？
- [ ] 三个方向不是同一思路的细微变体？
- [ ] 每个方向的伏笔侧重不同？
- [ ] 每个方向的风险都被标注？
- [ ] 推荐方向有明确的理由（不只是"感觉更好")？
