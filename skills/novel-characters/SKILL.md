# Novel Characters Skill

帮助创作者设计有深度、有个性、有动机的角色，构建人物之间的关系网络。

## 核心职责

1. **主角设计** - 创造有魅力、有成长弧光的主角
2. **配角群像** - 设计丰富立体的配角阵容
3. **反派塑造** - 创造有动机、有威胁的 antagonist
4. **关系网络** - 构建角色间的关系图谱
5. **角色一致性** - 确保角色行为符合其设定

## 工作流程

### 1. 确定角色需求

首先了解故事需要什么样的角色：

- 故事的主要冲突是什么？
- 主角需要什么样的特质来应对这个冲突？
- 需要哪些配角来辅助/阻碍主角？
- 需要什么样的反派来制造冲突？

### 2. 从功能出发设计

每个角色都应该有明确的叙事功能：

```markdown
角色: [名称]
功能: [在故事中承担的作用]
必要特质: [实现功能需要的性格/能力]
```

### 3. 构建角色深度

为角色添加层次：

- **表层**: 外貌、职业、技能
- **中层**: 性格、价值观、目标
- **深层**: 恐惧、创伤、隐藏欲望

### 4. 建立角色关系

设计角色间的化学反应：

- 角色A如何影响角色B？
- 他们之间的张力来源是什么？
- 关系如何随故事发展？

### 5. 检查角色平衡

确保角色阵容合理：

- 角色是否过于相似？（差异化检查）
- 是否有足够的冲突和合作？
- 配角是否有自己的故事线？

## 角色设计原则

### 主角设计

**有效主角的特质：**
- 有明确的目标和动机
- 有主动行动的能力
- 有成长的空间
- 有让读者共鸣的特质
- 有合适的"代价"（需要付出什么）

**主角弧线类型：**
1. **成长型** - 从弱小到强大
2. **堕落型** - 从光明到黑暗
3. **觉醒型** - 发现真相/自我
4. **救赎型** - 从错误中恢复

### 配角设计

**配角的功能：**
- **盟友** - 支持主角，提供资源/情感支持
- **导师** - 传授知识，引导成长
- **喜剧 relief** - 调节气氛
- **对比** - 通过反差突出主角特质
- **催化剂** - 推动剧情发展

**配角深度原则：**
每个配角都应该相信自己是自己故事的主角。

### 反派设计

**有效反派的特质：**
- 有合理的动机（从ta的视角看是正当的）
- 有足够的能力威胁主角
- 与主角有某种联系或相似性
- 有人性化的一面

**反派类型：**
1. **直接反派** - 主动对抗主角
2. **环境反派** - 自然灾害/制度/社会
3. **内在反派** - 主角自身的缺陷/心魔
4. **神秘反派** - 身份不明的威胁

## 引导问题

**主角：**
- "主角最想要什么？为什么？"
- "主角最大的恐惧是什么？"
- "主角有什么缺点/弱点？"
- "主角会为什么事情违背原则？"
- "主角的成长弧线是什么？"

**配角：**
- "这个配角在故事中起什么作用？"
- "配角与主角是什么关系？"
- "配角自己的目标是什么？"
- "配角会如何影响主角的旅程？"

**反派：**
- "反派为什么要这么做？"
- "反派认为自己是对的吗？"
- "反派与主角有什么联系？"
- "反派的弱点是什么？"
- "如何让反派既有威胁又有人性？"

## 技能调用

**何时调用：**
- 用户说"设计主角"、"创建角色"
- 用户需要设计配角阵容
- 用户需要创作反派
- 用户需要梳理人物关系
- 世界观完成后，进入大纲创作前

**前置条件：**
- 项目已初始化
- 建议先完成世界观（可选但推荐）

**后置操作：**
- 可与 novel-world 并行进行
- 完成后可进入 novel-outline

## 注意事项

1. **避免完美** - 完美角色无趣，要有缺陷
2. **避免平面** - 角色要有深度和层次
3. **避免工具人** - 配角要有自己的动机
4. **保持一致** - 角色行为要符合设定
5. **留有余地** - 不要一开始就把角色定死
